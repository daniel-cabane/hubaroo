## New feature

# Main idea
Add real time notification to the app

# Tech
Go online to find and study the for Laravel Reverb and Laravel Echo and how they work together. Take the time to be sure to understand their logic and best practices.
Once you know enough, install Laravel Reverb and Laravel Echo and configure them to achieve real time communication between back end and front end

# Real time logic
Add real time notifications for the following features : 
- On the session details view, the attempts should update in real time (update for each new answer)
- Teacher should see students joining his class in real time
- Students belonging to a class should see any new session opened for the class in real time
- When a session expires, all attempts in progress should be submitted
