## New feature

# Main idea
A logged in user should be able to see any paper (without the answers).

# Implementation
In the profile menu, add a link which opens a modal where the user can select a year and a level and confirm his choice. He is then taken a new view where the paper's questions are all displayed in one column.
At the top of the page, are the two select (year and level) if the user wants to navigate to another paper.
The page has buttons to lead him back or to create a session with this paper.
